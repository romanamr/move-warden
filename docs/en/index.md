# MoveWarden (EN)

Go CLI to move and rename files using declarative JSON rules.

## What you will find

- Quick start to run the app.
- `rules.json` format.
- Transformation and filter rules.
- Dynamic placeholders (`{filename}`, `{ext}`, etc.).
- Ready-to-adapt examples.
