# MoveWarden (ES)

CLI en Go para mover y renombrar archivos con reglas declarativas en JSON.

## Qué encontrarás aquí

- Inicio rápido para ejecutar la app.
- Formato de `rules.json`.
- Reglas de transformación y filtros.
- Variables dinámicas (`{filename}`, `{ext}`, etc.).
- Ejemplos listos para adaptar.
